리소스 관리

- 다양한 값을 코드에 직접 작성하지 않고 xml 을 통해 관리
- xml을 통해 단말기의 환경에 따라 대응
- 유지 보수에 용이
- 문자열 관리, 다국어 지원 가능



크기 관리

- 크기를 xml에 등록해서 사용 가능
- 디스플레이 장비는 px 단위로 크기를 표시

- 단말기에 따라 px 단위를 사용하면 크기가 다르게 표시됨



단위

- px 
  - 실제 사용할 픽셀의 개수
- dp
  - 160ppi 액정에서 1dp == 1px
- sp
  - 단말기에 설정되어 있는 글자 크기에 따라 변함
  - 기본 크기 160ppi에서 1sp ==1px
- pt
  - 1pt == 1/72인치
